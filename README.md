## Backend Setup

Navigate to the `backend` folder and run the following commands:

```bash
cd backend
npm install
node server.js
```

To run the test suite, run  `npm test`.

## Frontend Setup

Navigate to the `frontend` folder and run the following commands:

```bash
cd frontend
npm install
npm start
```
